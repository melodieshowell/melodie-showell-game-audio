Wwise Combat VO System

Video walkthrough of combat system in Wwise, running via a custom WAAPI controller. VO recorded by amateur actors from their home studios.

  • WAAPI Controller — Python/Tkinter tool controlling Events, RTPCs & Switches externally via ak.soundengine calls
  • Character Type — Switch Container, Switch Group: Character_Type, mutually exclusive
  • Attack Level — Switch Container per HP band, Switch Group: Attack_Level, Light/Heavy containing Random Container shuffling takes
  • Player HP — Blend Container per type, Game Parameter: Player_HP, three ranges on continuous value 1-100
  • Ducking — Player_Dialogue bus ducks Combat_Effort bus automatically on playback
  • Processing — Werewolf: pitch & formant shift varied per take, EQ, compression, LUFS matching on all files
